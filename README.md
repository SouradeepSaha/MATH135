# MATH135

This project contains frequently used function in the UW MATH135 course. The functions available are: 
- ```EEA(num1, num2)``` calculates the GCD of num1 and num2 Extended Eucleadean Algorithm
- ```LCT(a, b, m)```
- ```pubKey(p,q,e)```
- ```priKey(p, q, e)```
- ```powerMod(base, exp, m)```
